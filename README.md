<h2>ℹ️ Como utilizar</h2>

Clone esse repositório

$ git clone https://github.com/guilhermebarrosdev/Curso-React-Ts-MaterialUi.git

<hr>

Instalar dependências

$ yarn install

  <hr>

Inicie o projeto

$ yarn dev
$ yarn mock
(Simultâneos em abas diferentes no terminal)
